Constructor

new PxgridRestClient(pxGridControl)

Source:
  • pxgrid-client.js, line 80
See:
  • PxgridControl
  • Cisco PxGrid 2.0 GitHub Wiki for more information on the Cisco PxGrid 2.0 implementation.

The web sockets use a STOMP-based messaging framework.

Examples

PxgridClient instance setup excluded.

const PxgridRestClient = require('pxgrid-client');

const pxgrid = new PxgridControl(pxgridControlOptions);
const pxclient = new PxgridRestClient(pxgridControlObject);

pxgrid.isActivated()
  .then(() => {
    pxclient.getProfiles()
       .then(profiles => console.log(profiles));

    pxclient.applyAncToEndpointByMac('QUARANTINE', '11:00:00:00:00:01')
      .then(response => console.log(response));
  });

Example including the PxgridClient instance setup.

const fs = require('fs');
const PxgridControl = require('../lib/pxgrid-control');
const PxgridRestClient = require('../lib/pxgrid-client');

certs = [];
certs.certPath = './certs/';
certs.clientCert = fs.readFileSync(certs.certPath + 'publiccert.cer');
certs.clientKey = fs.readFileSync(certs.certPath + 'key.pem');
certs.caBundle = fs.readFileSync(certs.certPath + 'caBundle.cer');

const pxgridControlOptions = {
  host: 'my-ise-server.domain.com',
  client: 'node-pxgrid',
  clientCert: certs.clientCert,
  clientKey: certs.clientKey,
  caBundle: certs.caBundle,
  clientKeyPassword = false,
  secret = '',
  port = '8910'
}

const pxgrid = new PxgridControl(pxgridControlOptions);
const pxclient = new PxgridRestClient(pxgridControlObject);

pxgrid.isActivated()
  .then(() => {
    pxclient.getProfiles()
       .then(profiles => console.log(profiles));

    pxclient.applyAncToEndpointByMac('QUARANTINE', '11:00:00:00:00:01')
      .then(response => console.log(response));
  });

connectToBroker(debugopt) → {Promise}

Source:
  • pxgrid-client.js, line 202

Creates a STOMP client over a Web Socket connection to the PxGrid Controller. This returned client object can be passed to subscribe/publish function to enable pub/sub functionality.

Returned client must be activated prior to use using client.activate(). This can take a short period of time, so a setTimeout may be required to ensure client was activated before trying to use.

This may change in the future for better usage (e.g. wait period executed within connectToBroker()).

Example
const ancCallback = function (message) {
  const body = JSON.parse(message.body);
  console.log("Endpoint " + body.macAddress + " has had an " + body.status + " ANC event");
}

function main() {
  pxclient.connectToBroker()
    .then(session => {
      session.activate();
      setTimeout(() => {
        pxclient.subscribeToAncPolicies(session, ancCallback);
      }, 1500);
    });
}

pxgrid.isActivated()
  .then(() => main());
